## 📖 Books (Free or with Free Access)

### Must-Read

| Book | Author | Free Access | Focus |
|---|---|---|---|
| **Hands-On ML with Scikit-Learn, Keras & TensorFlow** (3rd ed.) | Aurélien Géron | [Notebooks free on GitHub](https://github.com/ageron/handson-ml3) | The best practical ML book. Period. |
| **Deep Learning** | Goodfellow, Bengio, Courville | [Free online](https://www.deeplearningbook.org/) | Reference for theory. Read chapters 1-8 |
| **Mathematics for Machine Learning** | Deisenroth, Faisal, Ong | [Free PDF](https://mml-book.github.io/) | Math foundations for ML |
| **An Introduction to Statistical Learning (ISLR)** | James, Witten, et al. | [Free PDF](https://www.statlearning.com/) | Accessible intro to statistical ML |

### Good Supplementary Reads

| Book | Author | Free Access | Focus |
|---|---|---|---|
| **Designing Machine Learning Systems** | Chip Huyen | No (but worth buying) | Production ML — the best book on real-world ML |
| **Think Stats** | Allen Downey | [Free online](https://greenteapress.com/thinkstats2/) | Stats with Python code |
| **Think Bayes** | Allen Downey | [Free online](https://greenteapress.com/wp/think-bayes/) | Bayesian thinking with code |

## 🎯 Recommended Learning Path

1. **Week 1-2:** 3Blue1Brown (Linear Algebra + Calculus) + StatQuest (Probability basics)
2. **Week 3-6:** Andrew Ng's ML Specialization (audit) — watch + code along
3. **Week 7-8:** Google ML Crash Course — solidify fundamentals
4. **Week 9-12:** Hands-On ML book (chapters 1-8) + build Project 1 & 2
5. **Week 13-16:** fast.ai deep learning course + build Project 3
6. **Week 17-20:** MLOps Zoomcamp + build Project 4
7. **Week 21-24:** Interview prep (questions + mock interviews)
