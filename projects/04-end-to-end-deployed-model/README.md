<![CDATA[# Project 4: End-to-End Deployed ML Model

## 🎯 Objective
Build, containerize, and deploy an ML model as a REST API. This is the project that separates you from 90% of other candidates.

## 📊 Suggested Problem
**Spam Email Classifier** — text classification is clean, demonstrable, and interview-friendly.

## 📋 Deliverables Checklist

- [ ] **Model training** — TF-IDF + Logistic Regression (or fine-tuned DistilBERT)
- [ ] **Model serialization** — save model + vectorizer with `joblib`
- [ ] **FastAPI app** — `/predict` endpoint with Pydantic validation
- [ ] **Health check** — `/health` endpoint
- [ ] **Unit tests** — test API with sample inputs (`pytest`)
- [ ] **Dockerfile** — containerize the application
- [ ] **docker-compose.yml** — one-command startup
- [ ] **CI/CD** — GitHub Actions: lint → test → build Docker image
- [ ] **Documentation** — API docs (auto-generated by FastAPI at `/docs`)
- [ ] **README** — architecture diagram, API examples with curl, how to run

## 🏗 Suggested Structure

```
04-deployed-model/
├── app/

│   ├── model.py           # Model loading and prediction
```
