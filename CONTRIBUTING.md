<![CDATA[# Contributing

Thank you for helping improve this roadmap! Here's how you can contribute.

## What We're Looking For

- **Corrections** — factual errors, broken links, typos.
- **Better resources** — a higher-quality free course or book that replaces an existing recommendation.
- **New interview questions** — real questions you've encountered in ML interviews.
- **Project improvements** — better starter templates, clearer instructions.

## What We're NOT Looking For

- Research-level or PhD-focused content.
- Promotional links to paid courses or products.
- Content that doesn't directly help with interview preparation.
- Surface-level "top 10 algorithms" additions.

## How to Contribute

1. **Fork** this repository.
2. Create a **feature branch**: `git checkout -b fix/typo-in-cnn-doc`
3. Make your changes.
4. Submit a **Pull Request** with a clear description of what you changed and why.

## Style Guide

- Use clear, professional language.
- Markdown formatting should be consistent with existing files.
- Tables should use the `| header |` format used throughout the repo.
- Code examples should be minimal, runnable, and well-commented.
- Avoid jargon without explanation.

## Code of Conduct

Be respectful, constructive, and focused on helping students succeed. That's it.
